## 🎯 What's Inside Each File

Every topic file includes:

- **Categorized problem tables** — problems grouped by sub-pattern (e.g., Monotonic Stack, Interval DP, 0-1 BFS)
- **LeetCode numbers** — direct reference to problem IDs
- **Difficulty ratings** — Easy / Medium / Hard
- **Study strategy** — recommended learning order for each topic
- **Senior/Staff priorities** — which patterns matter most for senior-level interviews
- **Code templates** — reusable templates for common patterns (sliding window, monotonic stack, BFS, etc.)
- **Common interview traps** — pitfalls that trip up candidates

## 🏗️ How to Use

1. **Pick a topic** based on your weakest area
2. **Start with Category 1** in that file — they're ordered by difficulty
3. **Master the templates** at the bottom of each file before grinding problems
4. **Focus on Senior/Staff priorities** if preparing for L5+ interviews
5. **Track your progress** — aim for 3-5 problems per sub-pattern

## ⭐ Key Patterns for FAANG Interviews

If you're short on time, prioritize these high-frequency patterns:

| Pattern | Topic File | Why It Matters |
|---------|-----------|----------------|
| Sliding Window (variable size) | Two Pointers | Tested in ~30% of array interviews |
| Monotonic Stack | Stack & Queue | Histogram, subarray contribution |
| BFS/DFS on Grids | Graphs | Most common graph problems |
| Binary Search on Answer | Binary Search | "Minimize maximum" pattern |
| Subsequence DP | Dynamic Programming | LCS, LIS, Edit Distance family |
| Interval Scheduling | Greedy | Meeting rooms, activity selection |
| Level-Order Traversal | Trees | Most common tree pattern |
| Top-K / Merge-K | Priority Queue | Heap fundamentals |
| Prefix Trie + DFS | Tries | Word search, autocomplete |
| Prefix Sum + HashMap | Arrays | Subarray sum patterns |
